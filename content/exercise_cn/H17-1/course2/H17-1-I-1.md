## 问1

设 a，b 为常数。x 的整式

\[
A = x^3 + ax^2 + 18,\qquad B = x^2 - 2x + b
\]

思考。

### （1）

当 A 用 B 除时，商记为 Q，余数记为 R，

\[
Q = x + a + \boxed{A}
\]

\[
R = (\boxed{B}a - b + \boxed{C})x - (a + \boxed{D})b + \boxed{EF}
\]

是。

### (2)

当 A 能被 B 整除时，a 和 b 的值是

\[
a = \boxed{G},\quad b = \boxed{H}
\]

或

\[
a = \boxed{IJ},\quad b = \boxed{KL}
\]

是。
