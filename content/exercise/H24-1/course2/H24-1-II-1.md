## II

初項から第 \(n\) 項までの和が

\[
\sum_{k=1}^{n} a_k = n^2 + 3n
\]

である数列 \(\{a_n\}\)（\(n = 1, 2, 3, \ldots\)）を考える。

(1)　\(a_n = \boxed{A}n + \boxed{B}\) である。

(2)　\(b_n = n^2 - 5n - 6\) である数列 \(\{b_n\}\)（\(n = 1, 2, 3, \ldots\)）に対して，  
\(b_n < 0\) となる項は全部で \(\boxed{C}\) 個あり，  
それらの項の和は \(\boxed{DE}\) である。

(3)　(1)，(2) の \(a_n, b_n\) に対して

\[
\sum_{k=1}^{n} \frac{k^2 b_k}{a_k}
= \frac{1}{\boxed{F}}\,n(n + \boxed{G})(n^2 - \boxed{H}n - \boxed{I})
\]

である。
