## III

正の数 \(x, y\) が

\[
(\log_2 x)^2 + (\log_2 y)^2 = \log_2 \frac{8x^2}{y^2} \ \ \ ……①
\]


を満たしながら変わるとき，\(xy^2\) の最大値および  
そのときの \(x, y\) の値を求めよう。

(1)　① の右辺は

\[
\log_2 \frac{8x^2}{y^2}
= \boxed{A}\log_2 x - \boxed{B}\log_2 y + \boxed{C}
\]

と変形できる。

したがって，\(\log_2 x = X,\ \log_2 y = Y\) とおくとき，  
① は \(X, Y\) を用いて

\[
(X - \boxed{D})^2 + (Y + \boxed{E})^2 = \boxed{F} \ \ \ \  ……②
\]


と表せる。

(2)　\(\log_2 xy^2 = k\) とおく。この式は (1) の \(X, Y\) を用いて

\[
X + \boxed{G}Y - k = 0 \ \ \ \  ……③
\]


と表せる。

ここで，\(XY\) 平面を考えると，② のグラフは円，③ のグラフは直線となる。  
\(k\) が最大になるのは，その円と直線が接するときである。

よって，\(k = \boxed{H}\) のとき，\(xy^2\) は最大値 \(\boxed{IJ}\) をとる。  
また，このとき \(x = \boxed{K},\ y = \boxed{L}\) である。
