問1　2つの放物線

ℓ：
\[
y=ax^2+2bx+c
\]

m：
\[
y=(a+1)x^2+2(b+2)x+c+3
\]

を考える。点 A, B, C, D が右図のような位置関係にあるとする。このとき，この 2 つの放物線のうち，一方は，3 点 A, B, C を通り，もう一方は，3 点 B, C, D を通るとする。

(1)　3 点 A, B, C を通る放物線は \(\boxed{A}\) である。ただし，\(\boxed{A}\) には，次の ⓪ か ① のどちらか適するものを選びなさい。

⓪　放物線 ℓ  
①　放物線 m

(2)　2 つの放物線 ℓ, m は，どちらも 2 点 B, C を通るので，点 B, C の x 座標は，2 次方程式
\[
x^2+\boxed{B}x+\boxed{C}=0
\]
の解である。よって，点 B の x 座標は \(\boxed{DE}\)，点 C の x 座標は  \(\boxed{FG}\) である。


(3)　特に，AB＝BC，CO＝OD のとき，a, b, c の値を求めよう。  
2 点 C, D は y 軸に関して対称であるから，\(b＝\boxed{H}\) である。また，AB＝BC より，直線 \(x＝\boxed{IJ} が \boxed{A}\) の軸である。したがって，\(a＝−\boxed{K}/\boxed{L}\) である。よって，
\[
c=\boxed{M}/\boxed{N}
\]
である。

注）対称：symmetry
